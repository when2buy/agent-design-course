### Bash is all you need：从 0 到 1 构建微型 Claude Code

在这个专题中，我们将基于开源项目 `learn-claude-code`，探索如何构建一个类似 Claude Code 的微型智能体 [1, 2]。该项目的核心哲学非常纯粹：“模型即智能体，我们的工作是为它提供工具并让其自由发挥（The model is the agent. Our job is to give it tools and stay out of the way）” [3]。

#### 核心模式 (The Core Pattern)

贯穿整个构建过程的**核心模式**是：每一次会话（Session）都会在基础循环之上叠加一种新机制，但**绝不改变核心循环本身** [4]。

#### 12步进阶学习路径

该项目包含了 12 个渐进式的会话，从最简单的循环开始，一路演进到完全隔离的自主执行状态 [5]。每一个会话都引入一个新机制，而每一个机制都对应一句核心的箴言（Motto） [5]。为了便于理解，我们将这 12 个会话划分为四个主要阶段：

1.  **第一阶段：基础循环与工具（s01 - s03）**
2.  **第二阶段：上下文与知识管理（s04 - s06）**
3.  **第三阶段：任务编排与后台处理（s07 - s08）**
4.  **第四阶段：多智能体团队协作（s09 - s12）**

后续的四篇文章将逐一为您深入拆解这四个阶段的内部运作机制。本指南在教学设计上非常克制，有意省略了部分复杂的生产级权限、完整的生命周期控制以及完整的 MCP（模型上下文协议）运行时细节，以便让学习者专注于智能体最核心的工作原理 [4]。
